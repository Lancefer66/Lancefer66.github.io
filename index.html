Current Project

Privacy-first semantic image renamer (Python).

⬇ Download AverageColor.exe

How It Works

  • Windows executable. No installation required. Simply place on desktop.
  • Drag any image onto the program icon.
  • The tool calculates the average RGB color of the image.
  • It renames the file by adding the RGB value to the front.
  • The original filename is preserved after a dash (-).
  • You can rename multiple files at once, may take almost a second for each file.
  • Once complete, simply hit F5 to resort directory alphabetically.
  • I have tested this on Windows 10, and with JPG files.
